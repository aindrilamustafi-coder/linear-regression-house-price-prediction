# House Price Prediction using Linear Regression

This project predicts house prices based on:
- Area
- Number of Bedrooms

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm
Linear Regression

## Features
✔ Predict house prices  
✔ Calculate Mean Absolute Error  
✔ Calculate R² Score  
✔ Visualize predictions using graph

## Dataset
Sample dataset containing:
- Area
- Bedrooms
- Price

## Output
Prediction values and graph visualization
